# Temwa Refilwe Nyirenda — Portfolio

Personal portfolio website built with HTML, CSS, and vanilla JavaScript. No frameworks, no build tools.

## Structure

```
Refilwe/
├── index.html
├── styles/
│   └── styles.css
└── assets/
    └── CV_TR_Nyirenda.pdf
```

## Features

- Responsive — mobile and desktop
- Dark and light mode via system preference
- Smooth scroll and fade-in animations
- CV downloads directly from the assets folder

## Getting Started

Open `index.html` in any modern browser. No setup needed.

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts — Jost, DM Mono
